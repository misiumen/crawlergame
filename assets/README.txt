DUNGEON KRAULEM — Assets
========================

audio/sfx/<key>.wav        — one-shot sound effects
audio/music/<key>.wav      — background music tracks
icons/<key>.png            — small UI icons (32x32, optional)

Accepted extensions:
    SFX:    .ogg  .wav
    Music:  .ogg  .mp3  .wav
    Icons:  .png  .jpg  .bmp  .gif


SHIPPED AUDIO (P29.13)
----------------------

The 11 SFX and 4 music tracks in audio/ are procedurally synthesized
chiptune — generated by dungeon_kraulem/tools/synthesize_audio.py from
stdlib (math + wave + struct, no external deps). Re-generate with:

    python -m dungeon_kraulem.tools.synthesize_audio

Every file is mono 16-bit 22050 Hz. To replace one with hand-authored
audio: drop a same-named .wav (or .ogg for SFX, .ogg/.mp3 for music)
into the same directory. The audio module picks .ogg over .wav when
both exist.

SFX keys (matched 1:1 to play_sfx() call sites in engine/):
    attack_fumble      attack_miss        enemy_death
    floor_descent      hit_crit           hit_landed
    limb_broken        player_crit_hit    player_death
    player_hit         sponsor_chime

Music keys (matched to Game._music_key_for_state):
    menu       — title screen + slot picker
    explore    — STATE_PLAY
    victory    — STATE_VICTORY
    defeat     — STATE_DEFEAT
